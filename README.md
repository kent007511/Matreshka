# Matreshka
Магазин тканей 🧵

## О проекте
Современный магазин тканей, построенный на Next.js 14 и развернутый на Vercel.

## Технологии
- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- ESLint

## Быстрый старт
```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Развертывание
Проект автоматически разворачивается на Vercel при каждом push в main.
